HTML Anchor Tag <a>

The <a> tag is used to create hyperlinks in HTML. It allows users to navigate to different pages, download files, open emails, jump to sections on the same page, and more.

Syntax

<a href="URL">Link Text</a>

href – Specifies the destination URL

Link Text – The clickable text users see

Absolute vs Relative URLs

Absolute URL:

<a href="https://example.com/page.html">Visit Page</a>

Relative URL:

<a href="folder/page.html">Visit Local Page</a>

Purpose

Tag Purpose
<a> Creates a hyperlink using the href attribute

Examples

1. External Website

<a href="https://www.google.com">Visit Google</a>
Visit Google

2. Open in New Tab

<a href="https://openai.com" target="_blank">OpenAI</a>
OpenAI

3. Link to Section on Same Page

<a href="#contact">Go to Contact Section</a>
Go to Contact Section

4. Email Link

<a href="mailto:help@example.com">Email Us</a>
Email Us

5. Download File

<a href="file.pdf" download>Download PDF</a>
Download PDF

6. Tooltip with title

<a href="https://example.com" title="Click to open example">Hover me</a>
Hover me

7. Send tracking data with ping

<a href="https://example.com" ping="https://tracker.com/ping">Track Click</a>
Track Click

All target Values

Value Description
_blank Opens link in a new tab or window
_self Default. Opens in the same tab/window
_parent Opens in the parent frame
_top Opens in the full body of the window (removes frames)

Common Attributes of <a>

Attribute Description
href The URL the link goes to
target Where to open the linked document
title Text shown as tooltip on hover
download Prompts user to download the resource
id Used to link to sections on the same page
rel Defines relationship between current and linked page (e.g. nofollow, noopener)
ping URLs where browser sends POST requests when the link is clicked (for tracking)

Best Practice

✅ Use meaningful text:

<a href="https://news.com">Read Latest News</a>

❌ Avoid vague links:

<a href="#">Click here</a>
