HTML Image Tag <img>

The <img> tag is used to embed images into a web page. It's a self-closing tag that requires at least two attributes: src and alt.

Syntax

<img src="URL" alt="Description">

Purpose

Tag Purpose
<img> Displays an image using a file path or URL

Example Image: 100 Days Challenge

Absolute vs Relative URL

Absolute URL Example:

<img src="https://m.media-amazon.com/images/I/71O-Os4GilL._UF1000,1000_QL80_.jpg" alt="100 Days Challenge">

Relative URL Example:

<img src="images/100days.jpg" alt="100 Days Challenge">

Examples

1. Basic Image

<img src="https://m.media-amazon.com/images/I/71O-Os4GilL._UF1000,1000_QL80_.jpg" alt="100 Days Challenge">

2. With Width and Height

<img src="..." width="300" height="300" alt="100 Days Challenge">

3. With Tooltip (title)

<img src="..." alt="100 Days Challenge" title="100 Days UI Challenge">

4. Lazy Loading

<img src="..." loading="lazy" alt="100 Days Challenge">

5. Broken Link Example (Shows alt text)

<img src="invalid.jpg" alt="Image not found">

Common Attributes

Attribute Description
src The image path or URL
alt Alternative text if the image doesn't load (used by screen readers)
width Sets the width of the image
height Sets the height of the image
title Text shown when hovered over the image
loading Allows lazy loading (lazy, eager)
decoding Defines image decoding mode (async, sync, auto)
referrerpolicy Controls referrer info sent with request (no-referrer, etc.)

Best Practices

  • ✅ Use alt for screen readers and broken images
  • ✅ Use loading="lazy" to speed up page load
  • ✅ Define width and height to prevent layout shift
  • ✅ Optimize images for web performance
  • ❌ Avoid uncompressed or overly large images

Accessibility Tip

Always write clear alt text to describe the image for users with screen readers.
