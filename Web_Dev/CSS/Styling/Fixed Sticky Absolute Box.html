CSS Positioning Explained

This demo shows the difference between three CSS positioning types:

  • Sticky (orange bar) - Sticks to top when scrolling
  • Fixed (blue box) - Always stays in same screen position
  • Absolute (green box) - Positioned relative to its parent container
